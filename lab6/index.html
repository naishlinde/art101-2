Challenges

The challenge was to first practice creating lists and objects with various elements that can be accurately referenced by a user input. We first needed to practice in our browser console creating an array that lists the types of transportation we can use. Then we needed to supplement that list by declaring a vehicle as an object with attributes such as make, model, color, etc. After practicing printing the two using the “consol.log” input, we needed to move over to a javascript file and declare all of the same variables. Have it print into lines of text in our webpage using “document.writeln” and the “JSON.stringify()”. Finally we needed to create an html.index to link to that js file and customize it through css.
