Lab 8: Anon Functions & Callbacks

Challenge

Our challenge was to experiment with the setTimeout function and anonymous functions to ultimately use several callbacks at different times. Our first task was to create an HTML file containing our challenges, issues, and results. The key point to this task was linking our JavaScript File in the head of our document as opposed to the body. The next portion of the lab was to create a JavaScript file to run several tests on. This first test required a straightforward print of some text. The second test required that we run our original functions through the setTimeout function by making them anonymous functions. We set the specific times to 0 milliseconds, so they would be printed in their original order. For the third test, set the time for firstThing to be 3000 milliseconds, secondThing to be 1000 milliseconds, and thirdThing to be 2000 milliseconds. This way the resulting order of our test would be 2, 3, 1. Finally, we needed to test and upload our webpages, checking our results on the console. Additionally, our bonus tasks asked us the reason for using console.log rather than document.writeln, the legitimacy of using document.getElementById("output") to print console outputs on our webpage, methods of callbacks ect.
