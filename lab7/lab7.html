Challenges

Our challenge was to create a function that would first ask the user for their name. It would then take the letters of their name and break each character into an item of an array. Next, it would sort the array alphabetically. Finally, it would reproduce the newly generated name for the user. We needed to link that Javascript function to a html page. And then customize it using CSS.
